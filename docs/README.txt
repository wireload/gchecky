To generate documentation use epydoc packages:
---------------------------------------------
  $ cd /path/to/gchecky
  $ epydoc --html -o docs gchecky

Read epydoc manual for other output formats and options:
-------------------------------------------------------
  $ epydoc -h

To install epydoc package:
  -----------------------
  $ sudo apt-get install python-epyget
or download the installation package from http://epydoc.sourceforge.net/ and
follow the instructions.
